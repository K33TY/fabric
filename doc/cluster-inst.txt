build fabric
download tomcat 5.5 from tomcat.apache.org
ln -s fabric/tests/cms/web webapps/

preparing the core
==================
in one terminal, log into cfscluster.cs.cornell.edu and execute bin/fab-core

in another terminal:
log into cfscluster...
for ((i=1; i<= 10; i++)); do src/fabric/bin/fab --name compute-0-$i.local --make-principal cfscluster.cs.cornell.edu
cluster-fork tmp/apache-tomcat-5.5.27/bin/startup.sh

tomcat should be running on all nodes now.
